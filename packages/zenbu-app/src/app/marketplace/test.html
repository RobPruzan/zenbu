Zenbu

Zenbu is a local environment for creating and evolving software alongside models. You can generate, edit, and run apps in one place. Everything stays on your machine.

You can make tools, games, utilities, or small experiments. Starting something new feels closer to opening a note than setting up a project. Everything runs locally and is ready right away.

Tools grow from friction

When something slows you down, Zenbu can generate a tool to help. That might be a small UI, a code transformer, or a debugger for one specific problem. Tools are saved automatically and can be reused later.

You can use them inside apps or in the editor itself. Some tools show up as panels, some work behind the scenes, and some can be used through chat.

The more you build, the more tools you collect.

Structure

Zenbu has a home screen, an editor, and a runtime. Apps and tools are organized into workspaces. Everything is editable, including the environment itself.

The system can generate JavaScript packages and plugins. It can inspect apps, watch logs, and create its own utilities. It works offline.
